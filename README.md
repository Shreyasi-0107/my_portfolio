🌐 Shreyasi Saha — Portfolio Website
🔗 Live Website: https://shreyasi-portfolio.vercel.app/
A modern, responsive, and interactive personal portfolio website built using React + Vite, showcasing my work in Data Science, Data Analytics, and Data Visualization.

🚀 About the Project
This portfolio is designed to present my skills, projects, and experience in a clean and visually engaging way. It focuses on smooth user experience, modern UI design, and performance optimization.

✨ Features
🎨 Modern UI with custom typography (Vendura font)
🌈 Multi-color hero section (Data Science, Analytics, Visualization)
🧭 Structured navigation
(Hero → About → Education → Skills → Projects → Certificates → Internships → Contact)
🌀 Smooth scrolling & section snapping
🖱️ Custom cursor interactions
📱 Fully responsive design
⚡ Fast performance with Vite

🛠️ Tech Stack
Frontend: React.js, Vite
Styling: Tailwind CSS / CSS3
Animation: Framer Motion
Icons: Lucide React
Tools: Git, GitHub
